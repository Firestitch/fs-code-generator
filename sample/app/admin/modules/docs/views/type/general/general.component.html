Name Template Variables

Date-specific: relative to the Doc's create date.
  • {{ '{$doc->month_short}' }} 3-letter month name
  • {{ '{$doc->month}' }} full month name
  • {{ '{$doc->quarter}' }} Q1,Q2,Q3,Q4
  • {{ '{$doc->day_of_month}' }} eg: 23
  • {{ '{$doc->year}' }} eg: 2019
Context-Specific: Relative to Doc's Project.
  • {{ '{$project->name}' }} eg: "Some Project Name"
  • {{ '{$project->abbreviation}' }} eg: "SPN"
